Die **FRITZ!Box 6820 LTE** ist ein kompakter Router für Internet über **Mobilfunk (LTE / 4G)**.  
Statt DSL oder Glasfaser nutzt das Gerät einfach eine **SIM-Karte**, um eine Internetverbindung herzustellen.

Damit eignet sich der Router besonders für Orte, an denen **kein klassischer Internetanschluss verfügbar ist**.

Typische Einsatzbereiche sind:

- Wohnungen ohne DSL-Anschluss  
- **Ferienhäuser oder Wochenendhäuser**  
- mobile Arbeitsplätze  
- Internet als **Backup-Verbindung**  



## Internet über Mobilfunk (LTE)

Die FRITZ!Box 6820 besitzt ein integriertes **LTE-Modem** mit Unterstützung für **4G (LTE)** und **3G (UMTS)**.

Das bedeutet:

- Internet über **SIM-Karte**
- keine DSL-Leitung notwendig
- schnelle Einrichtung

Nach dem Einlegen der SIM-Karte verbindet sich der Router automatisch mit dem Mobilfunknetz.



## WLAN für Zuhause

Der Router bietet **WLAN N mit bis zu 450 MBit/s** im 2,4-GHz-Band.

Das reicht für typische Anwendungen wie:

- Surfen im Internet
- Streaming
- Home-Office
- Smart-Home Geräte

Mehrere Geräte wie **Smartphones, Tablets oder Laptops** können gleichzeitig verbunden werden.



## Gigabit-LAN Anschluss

Neben WLAN verfügt die FRITZ!Box auch über einen **Gigabit-LAN-Port**.

Damit können Geräte direkt per Kabel verbunden werden, zum Beispiel:

- Desktop-PC
- Smart-TV
- Spielekonsole
- Netzwerk-Speicher (NAS)

Eine Kabelverbindung sorgt oft für eine besonders stabile Verbindung.



## Kurz eingeordnet

Die **FRITZ!Box 6820 LTE** ist eine gute Wahl, wenn:

- **kein DSL oder Glasfaser verfügbar ist**
- Internet über **SIM-Karte** genutzt werden soll
- ein **kompakter LTE-Router** für Zuhause benötigt wird
- eine einfache Lösung für **Ferienhaus oder Zweitwohnung** gesucht wird

Der Router verbindet **Mobilfunk-Internet mit klassischem WLAN** und ermöglicht so eine schnelle Internetversorgung auch ohne festen Internetanschluss.
