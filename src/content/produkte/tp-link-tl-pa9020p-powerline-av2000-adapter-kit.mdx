# Internet über die Steckdose

Die Installation ist einfach:

1. Adapter per LAN mit dem Router verbinden  
2. Adapter in die Steckdose stecken  
3. zweiten Adapter in einem anderen Raum anschließen  

Damit steht dort sofort eine **LAN-Verbindung** zur Verfügung.

Über die **zwei Gigabit-LAN-Ports** können Geräte wie:

- Desktop-PC  
- Smart-TV  
- Spielekonsole  

direkt verbunden werden.

---

# Integrierte Steckdose

Ein Vorteil dieses Modells ist die **integrierte Steckdose**.

Dadurch geht keine Steckdose verloren und andere Geräte können weiterhin angeschlossen werden.

Das ist besonders praktisch, wenn Steckdosen knapp sind.

# Wann Powerline sinnvoll ist

Powerline kann besonders hilfreich sein, wenn:

- WLAN durch **dicke Wände blockiert wird**
- der Router **weit entfernt steht**
- eine **stabile LAN-Verbindung** benötigt wird
- ein **Homeoffice oder Gaming-Setup** aufgebaut wird

Viele Nutzer entscheiden sich für Powerline, wenn **WLAN im Altbau nicht stabil funktioniert**.

---

# Powerline vs WLAN Repeater

Ein **Powerline Adapter** kann sinnvoller sein als ein WLAN Repeater wenn:

- mehrere dicke Wände dazwischen liegen
- das WLAN Signal sehr schwach ist
- eine stabile LAN Verbindung benötigt wird

Ein **Repeater** ist dagegen einfacher, wenn nur ein einzelner Raum erweitert werden soll.
